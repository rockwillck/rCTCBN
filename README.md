# rCTCBN
An adaptation of the CT-CBN program for R
## Installation
`rCTCBN` can be installed from GitHub using
```R
# install.packages("devtools")
devtools::install_github("rockwillck/rCTCBN")
```
Run `?ctcbn` to learn how to use the function.
