# rCTCBN
An adaptation of the CT-CBN program for R
## Dependencies
### For Mac
```bash
curl -O https://mac.r-project.org/openmp/openmp-16.0.4-darwin20-Release.tar.gz
sudo tar fvxz openmp-16.0.4-darwin20-Release.tar.gz -C /
```
